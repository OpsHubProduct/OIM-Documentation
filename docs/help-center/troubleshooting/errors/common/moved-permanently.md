# Resolving `Server Error: (301) Moved Permanently`

## Description

When you encounter an error such as:

`Server Error: (301) Moved Permanently`

Perform the following steps:

1. Check the **proxy settings**.

2. Use the **Proxy Utility** to configure the proxy settings.

   **Path:**

   ```text
   <OpsHub Migrator for Microsoft Azure DevOps Installation Folder>\Other_Resources\Resources\ProxyUtility
   ```

3. Refer to the **Proxy Configuration** section in the installation prerequisites documentation for more details.
